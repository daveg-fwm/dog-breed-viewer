# Dog breed viewer SPA

## Stack

- Vite
- React
- TypeScript
- React Router
- Tanstack Query
- TailwindCSS
- Headless UI

## Getting Started

Vite was installed using node v24.13.1

### Installation

Install the dependencies:

```bash
yarn
```

### Development

Start the development server with HMR:

```bash
yarn dev
```

Your application will be available at `http://localhost:5173`.
